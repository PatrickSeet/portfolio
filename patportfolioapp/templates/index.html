About me

Experience

I've spent a few years as an Quality Assurance Engineer testing digital certificate issuance system for the Department of Defense. Then I've moved on to medical devices, a Project Lead for the firmware interface team. I've got my hands on quite a few pacemakers, defibrillators and neurological devices. Always fascinated by Cloud technologies and the fast paces startup world, I joined a young startup as a Software Engineer to design and implement REST API for mobile apps. My current adventure is Senior Support Engineer at an amazing company. Never stop learning!

Projects

Hometracker

A simple responsive app to help my brother keeps track of offers made on properties. Django, Python, Javascript, PostgreSQL, AJAX, Leaflet, Amazon S3, CSS and bootstrap. More features to come as I experiment with different libraries.

Go

BattleFit

An app that lets friends compare fitness data from various brands of fitness wearables. Winner of Hero City Hackathon best Health and Fitness category in 11/2014. I was responsible for API calls and backend design. Django, Python, Javascript, Bootstrap, and PostgreSQL.
